UI SYSTEM RULES (APPLY TO ALL SCREENS)

1. SPACING

* Use only: 4, 8, 12, 16, 24
* Maintain equal left and right padding
* No element should touch screen edges
* Maintain consistent vertical spacing between sections

2. CARDS

* padding: 16
* borderRadius: 16
* marginBottom: 16
* consistent shadow/elevation

3. LAYOUT

* Use flexbox properly:

  * alignItems: center
  * justifyContent: space-between
* Maintain balanced and symmetrical layouts

4. COMPONENTS

* Reuse components instead of rebuilding UI
* Standard components:

  * Card
  * Header
  * Row
  * Badge
  * Button

5. ALIGNMENT

* Icons and text must be aligned
* Avoid cramped or uneven spacing
* Maintain consistent horizontal and vertical rhythm

6. EDGE RULES

* No clipping of badges or elements
* Ensure proper spacing from screen edges

IMPORTANT:

* Follow these rules for every screen
* Maintain consistency across the entire app
* Do not use random spacing or layout values
